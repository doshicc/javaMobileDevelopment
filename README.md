# Консольная игра 

Консольная игра-стратегия, где игрок должен добраться до замка, избегая или побеждая монстров.

## 🎮 Описание игры

Вы - персонаж (😽), который находится в нижней части игрового поля. Ваша цель - добраться до замка (🏰), расположенного в верхней части поля. На пути вам встречаются монстры двух типов:
- **Обычные монстры** (👻) - проверяют ваши математические способности
- **Большие монстры** (👹) - вызывают на игру в "Камень-ножницы-бумага"
